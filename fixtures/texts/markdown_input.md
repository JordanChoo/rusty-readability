# Installation Guide

Follow these steps to install the application on your computer.

## Requirements

You need a modern operating system. Windows, macOS, and Linux are all supported. Make sure you have at least two gigabytes of free disk space.

## Steps

1. Download the installer from the official website.
2. Run the installer and follow the on-screen instructions.
3. Open the application and create your first project.

## Configuration

The default settings work well for most users. Advanced users can modify the **configuration file** located in the `settings` directory. See the [full documentation](https://example.com/docs) for details.

```json
{
  "theme": "dark",
  "language": "en"
}
```

If you encounter any problems, please contact our support team.
